# 1. 프로젝트 작업 범위

 Ahub 프로젝트의 산출 결과물은 다음과 같습니다.
- **요구사항 명세서** : 프로젝트의 요구사항을 정리하여 기능별로 분류해 둔 문서
- **와이어 프레임/스토리보드** : Visily를 사용, 예상 결과 화면을 산출한 문서
- **기능 설계서** : 구현할 기능들을 정리하여 컴포넌트별로 분류해 둔 문서  
- **화면 설계서** : 화면단에서 실제 구현한 기능들에 대한 설명을 이미지와 함께 정리한 문서

- **테스트 결과** : 테스트 실행 결과물(영상)

# 5. 향후 계획

## 로그인 기능
- Oauth 2.0 적용해서 삼자인증 방식 로그인 추가 예정
- 프로필 페이지에서 수정버튼 눌렀을 때 회원정보 변경할 수 있도록 구현
- 프로필 페이지에서 회원탈퇴 버튼 눌렀을 때 DB에서 회원정보를 삭제하도록 구현

## 페어 | 뉴스 정보
- 각 게시판에서 페이징 처리 구현
- 태그를 사용한 메뉴 및 검색 구현
- 게시글 등록, 수정, 삭제 페이지 구현

## 후기 | 정보 게시판
- 게시글 등록, 수정, 삭제 페이지 및 기능 구현
- 정렬 기능 구현
- 검색 기능 구현
- 각 게시판 페이징 처리 구현
- 댓글 등록 및 수정 구현

## 기타
- 구현하지 못한 페이지 구현
- 다양한 예외 처리 구현
- 백엔드 서버 고도화

# 6. 프로젝트 회고

| 이름 | 내용 |
| ----- | ----------|
|    백동현   | 벌써 세번째 프로젝트까지 끝났다니 시간이 되게 빠르다고 느껴졌습니다. 백엔드는 다뤄봤던 적이 있어서 조금은 익숙했지만 프론트는 처음이라 html/css 부터 vue까지 모든게 하나같이 다 어렵고 낯설었습니다. 그래서 굉장히 막막했지만 구글링과 팀원들의 도움을 받아 완성할 수 있었습니다. 프로젝트의 마무리가 다가올수록 욕심이 나서 이런저런 기능을 더 추가하고 싶었지만 시간이 없어서 기능들을 더 추가를 못한것이 굉장히 아쉬웠고 그래서 프로젝트가 끝나더라도 추가로 더 고도화를 진행해야겠다고 생각했습니다. 이번 프로젝트까지 하며 디비, 백, 프론트 모두 다 다뤄보게되었는데 이 경험들을 통해서 최종프로젝트는 좀 더 완벽하게 만들 수 있을것같다고 느꼈습니다. |
| 손세림 | text |
